Tree Growth Graphics Simulation

This research project explored the various algorithms and techniques used for the creation of digital images of trees and plant life. The project used the Java programming language and the Processing graphics application to implement concepts such as orthonormal basis, matrix transformation, voxel and object-oriented programming to create three-dimensional respresentations of five different species of trees.
